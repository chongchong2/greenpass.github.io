♿ 교통 약자 배려

기술 스택

Framework & Language

  • Flutter 3.32.2
  • Dart 3.6+

Architecture

  • Clean Architecture
  • Riverpod (State Management)
  • Dependency Injection (get_it, injectable)

주요 라이브러리

  • 네트워킹: dio, retrofit, http
  • 지도 & 위치: flutter_naver_map, geolocator
  • 로컬 저장소: shared_preferences, hive
  • 유틸리티: freezed, json_serializable, equatable

로드맵

Phase 1 (완료) ✅

  • 서울시 C-ITS 실시간 신호 연동
  • 대전/하남 시뮬레이션 지원
  • 지도 기반 UI 구현
  • 자동 지역 감지

Phase 2 (진행 중) 🚧

  • 앱 아이콘 및 스플래시 이미지 디자인
  • TestFlight / Google Play 베타 배포
  • 사용자 피드백 수집 및 반영

Phase 3 (예정) 📅

  • 백그라운드 위치 트래킹
  • 진동/소리 알림 (Accessibility)
  • 추가 지역 확장 (경기도, 부산 등)
  • Apple Watch / Wear OS 지원
