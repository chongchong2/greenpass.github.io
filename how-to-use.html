사용방법

몇초남았지 앱을 효과적으로 사용하는 방법을 안내합니다.

📥 1. 앱 다운로드 및 설치

iOS (아이폰)

  1. App Store에서 "몇초남았지" 검색
  2. "받기" 버튼을 눌러 다운로드
  3. 설치가 완료되면 앱 아이콘을 탭하여 실행

💡 참고: iOS 14.0 이상 필요

Android (안드로이드)

  1. Google Play 스토어에서 "몇초남았지" 검색
  2. "설치" 버튼을 눌러 다운로드
  3. 설치가 완료되면 "열기" 버튼을 눌러 실행

💡 참고: Android 8.0 (Oreo) 이상 필요

🚀 2. 첫 실행 및 권한 설정

Step 1: 위치 권한 허용

앱을 처음 실행하면 위치 접근 권한을 요청합니다.

"몇초남았지"가 위치에 접근하도록 허용하시겠습니까?

  • ✅ "앱 사용 중에만 허용" 또는 "항상 허용" 선택 (권장)
  • ❌ "허용 안 함"을 선택하면 앱을 사용할 수 없습니다

💡 왜 필요한가요? 주변 신호등 정보를 표시하려면 현재 위치를 파악해야 합니다.

Step 2: GPS 활성화

정확한 위치 파악을 위해 GPS를 켜주세요.

  • iOS: 설정 → 개인정보 보호 → 위치 서비스 → 켜기
  • Android: 설정 → 위치 → 켜기

🗺️ 3. 메인 화면 사용하기

지도 화면 이해하기

🟢 녹색 원형 마커

보행자 신호가 녹색(건너도 됨)인 횡단보도입니다.

  • 마커 안에 표시된 숫자는 잔여 시간(초)
  • 예: "25" → 25초 후에 빨간불로 바뀜

🔴 빨간색 원형 마커

보행자 신호가 빨간색(건너면 안 됨)인 횡단보도입니다.

  • 마커 안에 표시된 숫자는 잔여 시간(초)
  • 예: "45" → 45초 후에 초록불로 바뀜

📍 내 위치 버튼

지도 우측 하단의 버튼을 누르면 현재 위치로 이동합니다.

⚙️ 4. 설정하기

설정 메뉴 열기

지도 화면 우측 상단의 ⚙️ 설정 아이콘을 탭합니다.

지역 수동 설정 (선택 사항)

앱이 자동으로 지역을 감지하지만, 필요시 수동으로 변경할 수 있습니다.

  • 서울: 실시간 C-ITS 데이터 (가장 정확)
  • 대전: 시뮬레이션 (실제 신호와 오차 있을 수 있음)
  • 하남: 시뮬레이션 (실제 신호와 오차 있을 수 있음)

💡 팁: 대부분의 경우 자동 감지를 그대로 사용하는 것이 좋습니다.

💡 5. 사용 팁

🎯 정확한 위치

실내보다는 야외에서, GPS 신호가 잘 잡히는 곳에서 사용하면 더 정확합니다.

🔋 배터리 절약

백그라운드에서는 위치 추적을 하지 않으므로 배터리 소모가 적습니다.

📶 네트워크 연결

실시간 신호 정보를 받으려면 인터넷 연결(Wi-Fi 또는 모바일 데이터)이 필요합니다.

🚦 시뮬레이션 지역

대전/하남은 시뮬레이션이므로 실제 신호와 최대 ±10초 오차가 있을 수 있습니다.

🔄 지도 확대/축소

지도를 핀치(두 손가락)로 확대하면 더 많은 신호등을 볼 수 있습니다.

⏱️ 실시간 업데이트

서울 지역은 매 초마다 실시간으로 업데이트되어 가장 정확한 정보를 제공합니다.

❓ 6. 자주 묻는 질문 (FAQ)

신호등이 지도에 표시되지 않아요

다음 사항을 확인해주세요:

  • 위치 권한이 허용되어 있는지 확인
  • GPS가 켜져 있는지 확인
  • 인터넷 연결 상태 확인
  • 서울/대전/하남 지원 지역 내에 있는지 확인
  • 지도를 충분히 확대했는지 확인
잔여 시간이 정확하지 않아요

서울 지역: C-ITS 실시간 데이터를 사용하므로 매우 정확합니다. 오차가 있다면 인터넷 연결 상태를 확인해주세요.

대전/하남 지역: 신호 주기 기반 시뮬레이션이므로 실제 신호와 최대 ±10초 오차가 있을 수 있습니다.

배터리 소모가 많지 않나요?

아니요. 앱이 백그라운드에 있을 때는 위치 추적을 하지 않습니다. 앱을 사용하는 동안만 GPS와 네트워크를 사용하므로 배터리 효율적입니다.

다른 지역에서도 사용할 수 있나요?

현재는 서울, 대전, 하남만 지원합니다. 향후 경기도, 부산 등 다른 지역으로 확장할 예정입니다. 원하시는 지역이 있다면 피드백을 보내주세요!

개인정보는 어떻게 관리되나요?

위치 정보는 서버에 저장되지 않으며, 오직 주변 신호등 정보를 표시하는 용도로만 사용됩니다. 자세한 내용은 개인정보처리방침을 참고해주세요.
