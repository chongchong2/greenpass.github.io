개인정보처리방침

최종 업데이트: 2025년 12월 5일

1. 개인정보의 처리 목적

"몇초남았지"(이하 "앱")는 다음의 목적을 위해 개인정보를 처리합니다. 처리하고 있는 개인정보는 다음의 목적 이외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 「개인정보 보호법」 제18조에 따라 별도의 동의를 받는 등 필요한 조치를 이행할 예정입니다.

주요 처리 목적

  • 위치 기반 서비스 제공: 사용자 주변의 실시간 보행자 신호등 정보 표시
  • 서비스 품질 향상: 앱 사용 패턴 분석 및 오류 진단
  • 사용자 지원: 문의 응답 및 기술 지원

2. 수집하는 개인정보의 항목 및 수집 방법

2.1 수집하는 개인정보 항목

구분 수집 항목 수집 목적 보유 기간
필수 위치 정보 (GPS 좌표) 주변 신호등 정보 제공 앱 사용 중에만 임시 저장, 서버 미전송
자동 수집 기기 정보 (OS 버전, 앱 버전) 서비스 호환성 확인 로그 기록 시
자동 수집 오류 로그 앱 안정성 개선 수집 후 30일

2.2 수집 방법

  • 앱 사용 중 자동 수집 (위치 권한 동의 시)
  • 오류 발생 시 자동 로깅
  • 사용자가 이메일로 문의 시 직접 제공

3. 개인정보의 보유 및 이용 기간

앱은 개인정보의 수집·이용 목적이 달성된 후에는 해당 정보를 지체 없이 파기합니다.

🔒 중요: 위치 정보 처리 방식

  • 실시간 처리만: 사용자의 위치 정보는 기기 내에서만 처리되며, 외부 서버로 전송되지 않습니다.
  • 즉시 삭제: 앱 종료 시 모든 위치 정보는 자동으로 삭제됩니다.
  • 저장 안 함: 위치 기록은 어디에도 저장되지 않습니다.

4. 개인정보의 제3자 제공

앱은 원칙적으로 사용자의 개인정보를 외부에 제공하지 않습니다. 다만, 다음의 경우에는 예외로 합니다.

  • 사용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

외부 서비스 이용

앱은 다음의 외부 서비스를 이용하며, 각 서비스의 개인정보처리방침이 적용됩니다:

  • Naver Maps API: 지도 표시 (Naver 개인정보처리방침)
  • 서울시 교통 빅데이터 플랫폼: C-ITS 실시간 신호 데이터 수신 (공공 API, 개인정보 미전송)
  • 공공데이터포털: 횡단보도 정보 수신 (공공 API, 개인정보 미전송)

5. 개인정보의 파기 절차 및 방법

앱은 개인정보 보유 기간의 경과, 처리 목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체 없이 해당 개인정보를 파기합니다.

파기 절차

  • 이용자의 개인정보는 목적 달성 후 즉시 파기됩니다.
  • 법령에 따라 보존해야 하는 경우에는 해당 기간 동안 안전하게 보관 후 파기합니다.

파기 방법

  • 전자적 파일: 복구 및 재생되지 않도록 안전하게 삭제
  • 기록물, 인쇄물: 분쇄기로 분쇄하거나 소각

7. 개인정보의 안전성 확보 조치

앱은 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

  • 관리적 조치: 내부관리계획 수립·시행, 정기적 직원 교육
  • 기술적 조치: 개인정보처리시스템 등의 접근권한 관리, 암호화 기술 적용
  • 물리적 조치: 개인정보가 저장되는 기기의 안전한 관리

🔐 주요 보안 조치

  • 위치 정보는 기기 내에서만 처리되며 외부 전송 없음
  • HTTPS를 통한 안전한 API 통신
  • 최소한의 권한만 요청 (위치 정보만)
  • 오픈소스 라이브러리의 정기적인 보안 업데이트
